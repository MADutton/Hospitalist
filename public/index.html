Lesson (12–15 min)

Goal: define “hospitalist scope” so it’s clearly distinct from ER triage and primary ICU care.

1) What is a veterinary hospitalist?

  • Owns longitudinal inpatient care (hours → days), not initial triage or ventilator-level ICU management.
  • Prevents deterioration through structured monitoring, early recognition, and timely escalation.
  • Optimizes care transitions: ER → ward, ICU → step-down, ward → discharge.

2) Scope boundaries (ABVP-friendly language)

  • Included: inpatient medical management, step-down planning, ward-level oxygen/fluids/electrolytes, antimicrobial stewardship, discharge optimization.
  • Excluded: primary ER triage responsibility; primary ICU case ownership (ventilator/vasopressors/CRRT management).
  • Escalation is a core skill: recognizing when ward care is no longer appropriate.

3) The “time horizon” mental model

Emergency practice

Minutes → hours. Stabilize, triage, decide disposition.

Hospitalist practice

Hours → days. Reassess trends, prevent complications, coordinate care.

4) What “mastery” looks like (Week 1)

  1. Creates a prioritized inpatient problem list.
  2. Defines objective deterioration thresholds.
  3. Documents a monitoring plan that a team can execute.
  4. States when to escalate to ICU or refer to a specialist.

5) Common inpatient failure points

  • “Stable” patient not reassessed for trend changes (RR, mentation, weight, urine output).
  • Fluid creep → overload (especially CHF/CKD cats).
  • Electrolyte drift (K, Na) not anticipated.
  • Handoff gaps: ER note lacks monitoring plan or escalation triggers.

6) Documentation standard (one-liner)

“If the overnight team reads only one paragraph, they should know: the top problems, the monitoring plan, and the trigger to escalate.”
